How it works
  • 1 Image is converted to grayscale, then thresholded to pure black & white.
  • 2 Pixels brighter than the background threshold are made fully transparent.
  • 3 Black pixel runs are encoded as SVG <rect> elements — scalable, no raster data.
Python CLI — batch & higher quality

The companion Python tool uses potrace when available for smooth, truly scalable vector paths. Find it in the png-to-svg/ folder of the repo.

# Install dependencies pip install -r requirements.txt # Convert (output defaults to same filename with .svg) python converter.py photo.png # Custom thresholds & explicit output path python converter.py photo.png output.svg --threshold 100 --bg-threshold 220
